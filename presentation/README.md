This folder contains a simple 7-slide presentation as SVG files.

Files:
- slide1.svg — Title slide
- slide2.svg — Project overview
- slide3.svg — Dataset
- slide4.svg — Packages & environment
- slide5.svg — Method / Approach
- slide6.svg — Timeline / Deliverables
- slide7.svg — Conclusion & Next steps

To open: view the SVG files in a browser or an image viewer.
To convert to PowerPoint:
- In PowerPoint: Insert → Pictures → This Device, select the SVGs (one per slide).
To import into Google Slides:
- Create a new slide deck, then use Insert → Image → Upload from computer for each SVG.

If you want, I can convert these SVGs into a `.pptx` file programmatically and add it to the repo.
